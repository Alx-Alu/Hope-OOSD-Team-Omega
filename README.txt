Folder structure:

+ bin - Contains the compiled .DDL/ .lib/ .exe files ("Binary").
+ include - Contains public header files for a library.
+ lib - Contains statically linkable .lib files (Pre-Compiled).
+ doc - Contains relevant documentation for software.
+ src - Source code.
+ data - Test data used by software.
